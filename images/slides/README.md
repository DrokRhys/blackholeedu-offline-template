# Slide Images Setup

This folder is used by `blackholeedu` slide backgrounds.

## Required file names

Put your exported images exactly with these names:
source: https://cdn.esahubble.org/
source: https://science.nasa.gov/

- `slide-02.jpg` 
- `slide-03.jpg` 
- `slide-04.jpg` 
- `slide-05.jpg`
- `slide-06.jpg`
- `slide-07.jpg`
- `slide-08.jpg`
- `slide-09.jpg`
- `slide-fallback.jpg` (used when one slide image is missing/broken)

Slide `01` has no right-side image by design.

## Recommended image parameters

- Format: `JPG` (preferred) or `WEBP`
- Resolution: `1920x1080` minimum
- Orientation: landscape (`16:9` recommended)
- Color profile: `sRGB`
- Keep important subject on the right half (left side is strongly darkened by gradient)
- Avoid text/watermarks near center-left (can clash with slide content)
